# Instapro

MVP социальной сети для обмена фотографиями

## Что нужно:

1. интегрировать верстку списка постов с API
2. реализовать страницу добавления поста
3. реализовать страницу с постами конкретного юзера
4. реализовать функционал лайков

## Удалось реализовать:

* рендер постов из api
* api и страницу добавления поста
* api и страницу с постами конкретного юзера
* функционал лайков

## Дополнительные фичи, которые удалось реализовать:

* перерисовка формы загрузки изображения при проблемах с интернетом
